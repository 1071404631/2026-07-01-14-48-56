# 2026-07-03 工作日志

## Vercel 部署上线后的 Bug 修复

### 问题1：左侧已提交职位列表不显示
- 原因：`api/submissions.js` 中按姓名过滤时用 `r.submitterName`，但实际数据结构是 `r.submitter.name`
- 修复：改为 `r.submitter && r.submitter.name === name`

### 问题2：Logo 图片不显示
- 原因：`vercel.json` 使用旧版 `builds` + `routes` 配置，catch-all 路由干扰了静态文件服务
- 修复：改为新版 `rewrites` 配置，只重写 API 路由，静态文件（HTML/PNG）由 Vercel 自动服务
```json
{
  "rewrites": [
    { "source": "/api/submit", "destination": "/api/submit.js" },
    { "source": "/api/submissions", "destination": "/api/submissions.js" }
  ]
}
```

### 问题3：后台删除记录无反应
- 原因1：`admin.html` 发送 `DELETE /api/submissions/{id}` 路径参数，但 Vercel 函数不认这个路径
- 原因2：`api/submissions.js` 只处理了 GET，没有 DELETE 逻辑
- 修复：
  - `admin.html` 改为 `DELETE /api/submissions?id=xxx` 查询参数
  - `api/submissions.js` 增加 DELETE handler，从 JSONBin 读取全部记录，过滤掉指定 ID 后写回
